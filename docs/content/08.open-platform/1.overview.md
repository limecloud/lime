# 扩展接入概览

::alert{type="info"}
当前这组“扩展接入”文档只保留仍有实现锚点的高级接入说明，不再把插件市场、生态合作流转或未落地的扩展能力写成既成事实。
::

如果你是普通创作者，可以直接跳过本章节。这里面向的是需要扩展 Lime、分发接入包、或让外部系统与 Lime 建立连接的开发者。

## 当前保留能力

### 插件打包与安装

当前文档覆盖的是已经落地的插件安装链路：

- 本地文件安装
- URL 安装
- `plugin.json` 清单约束
- UI surface 与二进制字段的基础说明

[查看插件打包说明 →](/open-platform/plugin-development)

### Connect 接入

当前文档覆盖的是已经落地的 Connect 能力：

- `lime://connect` 深链写入连接参数
- relay 注册表校验
- 用户确认后保存
- 可选 webhook 回调

[查看 Connect 说明 →](/open-platform/connect)

- 需要把外部站点或服务接入 Lime 的开发者
- 需要对接 Connect 深链或回调的集成方
