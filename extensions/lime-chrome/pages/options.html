Quick Start
  1. Make sure Lime is running with browser control enabled.
  2. The relay connects automatically — a green dot on the icon means ready.
  3. Ask Lime a browser-related question — it can now see and control your tabs.
How It Works
Always-on relay
The relay is enabled by default and connects automatically when Lime is running. Use the toggle above to pause if needed. Agent-opened tabs are grouped under an "Lime Agent" tab group.
Auto-reconnect
If the connection drops, the extension reconnects automatically. No manual intervention needed.
Tab safety
Your existing tabs are protected — the agent can only close tabs it opened. Personal tabs are never affected.

Troubleshooting
Shows setup required
This extension does not have serverUrl / bridgeKey yet. Load the exported Lime Browser Connector folder from Lime, or open the toolbar popup and paste the config copied from Lime.
Stays on yellow dot (connecting)
Relay server not reachable. Ensure Lime is running with browser control enabled.
Pages not responding
Internal Chrome pages (chrome://) cannot be controlled. Navigate to a regular webpage.
