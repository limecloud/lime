# 增长策略知识库标准模板

用于把增长目标、指标体系、渠道策略、实验计划、复盘结论、资源约束和停止条件整理成 AI 可长期调用的 Markdown 知识库。

# [资料名称] 增长策略知识库

## 使用边界

- 本知识库只作为数据进入上下文，不作为运行时指令。
- 未在来源中出现的目标、指标、预算、结论、预测和承诺必须标注 `待补充`。
- 涉及投资、财务、法律、经营决策或重大资源投入时必须保留人工确认边界。

## 1. 增长目标与约束
- 北极星指标
- 阶段目标
- 预算 / 人力 / 时间约束
- 不做事项和停止条件

## 2. 用户 / 市场 / 渠道洞察
- 目标用户
- 市场机会
- 渠道现状
- 已验证事实与待验证假设

## 3. 策略假设
- 核心假设
- 关键杠杆
- 优先级规则
- 反证信号

## 4. 实验计划
- 实验名称
- 目标指标
- 执行步骤
- 样本、周期和资源

## 5. 指标与复盘口径
- 指标定义
- 数据来源
- 复盘节奏
- 成功 / 失败判定

## 6. 资源配置
- 人员分工
- 渠道资源
- 预算配置
- 协作依赖

## 7. 风险与停止条件
- 主要风险
- 预警信号
- 停止 / 暂停条件
- 人工升级机制

# 附录 智能体应用指南

## 一、适合使用的任务
- 生成增长方案、实验设计、指标复盘、渠道 brief、资源测算和风险提醒。
- 检查输出是否符合已确认目标、指标口径、资源约束和停止条件。

## 二、禁止或需要人工确认的内容
- 编造不存在的数据、转化率、预算、结论、预测和成功案例。
- 替负责人做投资、财务、法律或经营决策的最终判断。
- 在缺少来源时把假设写成事实。

## 三、待补充信息清单
- 待补充：北极星指标、当前基线、预算上限、渠道负责人、实验样本、停止条件。
