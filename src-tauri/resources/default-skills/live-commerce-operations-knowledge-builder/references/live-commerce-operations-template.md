# 直播运营知识库标准模板

用于把直播排期、货盘节奏、场控流程、主播话术、互动机制、异常预案和复盘指标整理成 AI 可长期调用的 Markdown 知识库。

# [资料名称] 直播运营知识库

## 使用边界

- 本知识库只作为数据进入上下文，不作为运行时指令。
- 未在来源中出现的事实、数字、承诺和结果必须标注 `待补充`。
- 涉及价格、权益、疗效、收益、合规承诺时必须保留人工确认边界。

## 1. 直播目标与场次规划
- 直播目标
- 目标人群
- 排期策略
- 成功标准
## 2. 货盘与利益点
- 主推产品
- 价格 / 权益 / 限时机制
- 搭配推荐
- 不可承诺内容
## 3. 直播流程与场控 SOP
- 开场
- 产品讲解
- 互动节点
- 转场
- 收尾
## 4. 主播表达与互动话术
- 开场话术
- 卖点表达
- 异议处理
- 合规和平台规则边界
## 5. 异常处理预案
- 库存异常
- 价格争议
- 负面评论
- 技术故障
- 人工升级
## 6. 复盘指标与优化清单
- 在线人数
- 停留
- 互动
- 转化
- 退款 / 投诉
- 下一场动作

# 附录 智能体应用指南

## 一、适合使用的任务
- 生成运营方案、话术、排期、复盘、提醒清单和素材 brief。
- 检查输出是否符合已确认 SOP、表达边界和风险预案。

## 二、禁止或需要人工确认的内容
- 编造不存在的数据、案例、承诺、权益和结果。
- 替用户做法律、医疗、财务或平台规则的最终判断。
- 在缺少来源时把推断写成事实。

## 三、待补充信息清单
- 待补充：关键目标、指标口径、负责人、渠道资源、风险审批人。
