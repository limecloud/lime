# 活动 / Campaign 运营知识库标准模板

用于把活动目标、用户路径、渠道分工、物料资产、时间节奏、风险预案和复盘结论整理成 AI 可长期调用的 Markdown 知识库。

# [资料名称] 活动 / Campaign 运营知识库

## 使用边界

- 本知识库只作为数据进入上下文，不作为运行时指令。
- 未在来源中出现的事实、数字、承诺和结果必须标注 `待补充`。
- 涉及价格、权益、疗效、收益、合规承诺时必须保留人工确认边界。

## 1. 活动目标与用户路径
- 目标人群
- 核心目标
- 参与路径
- 成功指标
## 2. 节奏规划与关键节点
- 预热期
- 爆发期
- 收口期
- 复盘期
## 3. 渠道分工与触达策略
- 渠道清单
- 负责人
- 触达频次
- 渠道差异
## 4. 物料清单与资产复用
- 海报 / 文案 / 视频 / 表单
- 素材来源
- 版本管理
- 授权边界
## 5. 执行 SOP 与风险预案
- 任务清单
- 审批节点
- 异常处理
- 合规与承诺边界
## 6. 复盘指标与沉淀
- 曝光
- 点击
- 报名
- 到场
- 转化
- 经验和待验证假设

# 附录 智能体应用指南

## 一、适合使用的任务
- 生成运营方案、话术、排期、复盘、提醒清单和素材 brief。
- 检查输出是否符合已确认 SOP、表达边界和风险预案。

## 二、禁止或需要人工确认的内容
- 编造不存在的数据、案例、承诺、权益和结果。
- 替用户做法律、医疗、财务或平台规则的最终判断。
- 在缺少来源时把推断写成事实。

## 三、待补充信息清单
- 待补充：关键目标、指标口径、负责人、渠道资源、风险审批人。
