# 品牌人设访谈问题

当来源资料不足时，优先向用户追问这些问题。

## 品牌定位

1. 品牌希望被一句话如何介绍？
2. 品牌最重要的 3 个价值观是什么？
3. 品牌明确反对什么？
4. 哪些表达听起来“不像这个品牌”？

## 受众关系

1. 核心受众是谁？他们为什么信任这个品牌？
2. 品牌与受众更像专家、朋友、陪伴者、顾问还是同行？
3. 哪些人群或场景不适合本品牌？

## 语气与内容

1. 请提供 3-5 篇最像品牌自己的内容样例。
2. 品牌可以幽默、强势、克制、温柔或犀利到什么程度？
3. 不同渠道是否需要不同语气？
4. 哪些词、句式、口号或内容结构最能代表品牌？

## 风险边界

1. 哪些承诺必须有证据或审批？
2. 哪些敏感问题必须转人工？
3. 出现投诉、质疑或舆情时，品牌第一反应原则是什么？
4. 价格、权益、功效、资质、结果类内容由谁最终确认？
