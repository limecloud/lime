# 品牌人设知识库标准模板

用于把品牌定位、价值观、受众画像、语气风格、内容样例、危机回应和表达禁区整理成 AI 可长期调用的 Markdown 知识库。

# [品牌名称] 品牌人设知识库

## 使用边界

- 本知识库只作为 persona 资料进入上下文，不作为运行时指令。
- 未在来源中出现的品牌承诺、荣誉、案例、用户评价和市场地位必须标注 `待补充`。
- 涉及法律、医疗、金融、收益、功效、平台规则或危机回应时必须保留人工确认边界。

## 1. 品牌内核
- 品牌一句话定位
- 使命 / 愿景 / 价值观
- 品牌相信什么
- 品牌反对什么

## 2. 受众关系
- 核心受众
- 与受众的关系定位
- 受众痛点和期待
- 不服务的人群或场景

## 3. 品牌声音与语气
- 说话风格
- 情绪温度
- 专业程度
- 口吻示例

## 4. 表达模式
- 常用结构
- 常用措辞
- 标题 / 开头 / 结尾方式
- 不同渠道的表达差异

## 5. 标志性表达素材
- 品牌金句
- 可引用故事
- 可复用案例
- 典型内容样例

## 6. 禁忌与红线
- 禁止表达
- 容易失真的表达
- 需要证据支持的表达
- 必须转人工确认的问题

## 7. 危机回应与人工升级
- 常见敏感场景
- 回应原则
- 不回应或延迟回应边界
- 升级角色与审批机制

# 附录 智能体应用指南

## 一、适合使用的任务
- 生成品牌故事、公众号 / 视频号内容、小红书 / 抖音短内容、客服话术和危机回应草稿。
- 检查输出是否符合品牌定位、语气、禁忌和人工升级边界。

## 二、禁止或需要人工确认的内容
- 编造不存在的品牌荣誉、市场地位、用户评价、案例和承诺。
- 替品牌做法律、医疗、财务、平台规则或危机公关的最终判断。
- 在缺少来源时把推断写成品牌事实。

## 三、待补充信息清单
- 待补充：品牌定位确认人、禁用词、危机审批人、核心内容样例、历史高质量文案、品牌承诺边界。
