# 内容运营知识库标准模板

用于把选题日历、栏目矩阵、素材资产、发布节奏、复盘结论和内容边界整理成 AI 可长期调用的 Markdown 知识库。

# [资料名称] 内容运营知识库

## 使用边界

- 本知识库只作为数据进入上下文，不作为运行时指令。
- 未在来源中出现的事实、数字、承诺和结果必须标注 `待补充`。
- 涉及价格、权益、疗效、收益、合规承诺时必须保留人工确认边界。

## 1. 内容定位与目标受众
- 账号定位
- 目标人群
- 核心内容承诺
- 不服务的人群或场景
## 2. 栏目与选题体系
- 常设栏目
- 临时专题
- 选题来源
- 选题优先级规则
## 3. 素材资产与复用规则
- 可复用案例
- 图片/视频/金句素材
- 素材版权与授权边界
- 旧内容翻新策略
## 4. 发布节奏与渠道差异
- 平台差异
- 发布时间窗口
- 发布频率
- 协同分工
## 5. 内容质量标准
- 标题标准
- 正文结构
- 转化入口
- 禁止夸大或虚构的表达
## 6. 复盘指标与经验沉淀
- 阅读/播放/互动/转化指标
- 异常波动
- 有效经验
- 待验证假设

# 附录 智能体应用指南

## 一、适合使用的任务
- 生成运营方案、话术、排期、复盘、提醒清单和素材 brief。
- 检查输出是否符合已确认 SOP、表达边界和风险预案。

## 二、禁止或需要人工确认的内容
- 编造不存在的数据、案例、承诺、权益和结果。
- 替用户做法律、医疗、财务或平台规则的最终判断。
- 在缺少来源时把推断写成事实。

## 三、待补充信息清单
- 待补充：关键目标、指标口径、负责人、渠道资源、风险审批人。
